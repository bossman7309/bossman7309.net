wbg                                                                version 1.0.2
________________________________________________________________________________

Super simple wallpaper application for Wayland compositors implementing the
layer-shell protocol. Wbg takes a single command line argument: a path to an
image file. This image is displayed scaled-to-fit on all monitors. [0]

Upstream: https://codeberg.org/dnkl/wbg
