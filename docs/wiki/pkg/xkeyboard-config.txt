xkeyboard-config                                                    version <a href=https://github.com/kisslinux/repo/tree/master/wayland/xkeyboard-config>2.34</a>
________________________________________________________________________________

The non-arch keyboard configuration database for X Window. The goal is to
provide the consistent, well-structured, frequently released open source of X
keyboard configuration data for X Window System implementations (free, open
source and commercial). The project is targeted to XKB-based systems. [0]

Upstream: https://www.freedesktop.org/wiki/Software/XKeyboardConfig/
